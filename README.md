# GLKLineKit
这可能是目前性能最高的K线图之一👍

## 特点：
	1. 高性能，处理1k条数据内存占用20~30M
	2. 高扩展性，采用插入算法的方式扩展功能
		2.1 添加一个新的指标只需要编写一个算法类，添加到数组中就可以了
		2.2 可以像调整图层一样调整指标组合
